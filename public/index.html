Si ya cuentas con un usuario registrado para acceder a nuestra aplicación web, por favor ingresa tus credenciales de usuario y contraseña para iniciar sesión.

Si aún no tienes acceso a la plataforma y deseas solicitarlo, te invitamos a seguir estos pasos:

  1. Envía un correo electrónico a helpdesk@gmail.com solicitando acceso a la aplicación web.
  2. Asegúrate de adjuntar el formulario de solicitud de acceso, el cual puedes encontrar en el siguiente enlace: Descargar formulario
  3. Nuestro equipo de soporte revisará tu solicitud y te proporcionará las instrucciones necesarias para completar el proceso de registro y acceso.

Para aclaraciones, comunicarse a la oficina de Informática.
